## 何をしているか

太陽熱養生（太陽熱で土壌を加熱する農業技術）の効果を定量的に検証するため、畑の複数地点に温度センサーを設置し、リアルタイムで計測・記録するシステムを自作しました。

| 項目 | 内容 |
|---|---|
| センサー | DS18B20 × 7本（深さ別・場所別） |
| マイコン | Raspberry Pi Pico W（MicroPython） |
| 通信 | MQTT（WiFi） |
| サーバー | Raspberry Pi 4 + Mosquitto + SQLite |
| 可視化 | Grafana |
| バックアップ | SDカード記録（30分ごと） |

---

## なぜ作ったか

太陽熱養生の効果は「地温が上がる」と言われているが、実際に何度まで上がるのか、深さによってどう違うのか、定量データがなかった。農業の話として「やってみたらよかった」で終わらせたくなかったので、ちゃんと計測する仕組みから自作した。

---

## システム構成

```
[PicoBox × 3台]
  └── DS18B20 × 7本
  └── SDカード（ローカル記録）
       │
       └── MQTT(WiFi) ──> [Raspberry Pi 4]
                               ├── Mosquitto（MQTTブローカー）
                               ├── Python logger → SQLite
                               └── Grafana（可視化）
```
